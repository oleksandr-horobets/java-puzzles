Java Puzzles
------------

Java Puzzles is a set of programming challenges to practice coding skills.

Each task goes with predefined set of tests.

Challenges
----------

* recursion
    - factorial
    - fibonacci
* arrays
    - array sum
    - linear search
    - binary search
    - find duplicates
    - merge sorted arrays
* lists
    - list sum
    - search in list
    - append element to list
    - reverse list
    - concat lists
    - merge sorted lists
* trees
    - tree depth
	- smallest number in tree
* sorting
    - selection sort
    - insertion sort
    - bubble sort
    - quick sort
	- merge sort
* strings
    - is palindrome
	- remove duplicate letters
* math
    - is prime
    - calculate square root
